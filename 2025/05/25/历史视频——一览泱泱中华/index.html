七下历史复习资料

第1课 隋朝统一与灭亡

【必背重点】

  1. 隋的统一：589年，隋文帝杨坚灭陈，完成统一，结束了魏晋南北朝以来长期分裂局面。
  2. 开创科举：隋炀帝杨广设置进士科，创立科举制。
  3. 科举作用：科举制度是我国古代选官制度的重大变革，延续1300多年，它加强了朝廷在选官上的权力，扩大了统治基础，促进了社会阶层流动，显著提高了整个官僚队伍的文化素养。
  4. 隋朝大运河：“尽道隋亡为此河，至今千里赖通波”反映的工程是隋朝大运河，“至今千里赖通波”反映出大运河加强南北地区政治、经济、文化交流的积极作用。
  5. 开凿运河：隋炀帝杨广为巩固统治，命人开凿了以洛阳为中心，北抵涿郡、南达余杭的大运河。
  6. 隋朝兴亡：581年，隋文帝杨坚建立隋朝，618年，因隋炀帝杨广暴政，隋朝灭亡。
  7. 秦隋对比：隋朝和秦朝一样如同流星一般辉煌而短暂，二者都完成大一统，都开创新制度，都开展大工程，都影响深远的同时又都短命的二世而亡。
  8. 兴亡启示：隋朝和秦朝都因暴政灭亡的历史启示我们，得民心者的天下，失民心者失天下，统治者应爱惜民力。

【扩展掌握】

  1. 开创盛世：隋文帝杨坚统一后励精图治，发展经济，编订户籍，统一币制，加强中央集权，简化机构，提高行政效率等，使得隋朝经济迅速恢复和发展，成为国力强盛的王朝。
  2. 制度创新：隋文帝杨坚开创了三省六部制，初步确立了通过考试选拔人才，依才能取士的制度，隋炀帝杨广设置进士科，正式确立科举制。
  3. 隋炀帝暴政：隋炀帝杨广急功近利，不恤民力，营建东都洛阳、开凿大运河，修筑长城和驰道，三征高句（gou）丽，使得民不聊生，最终引发大规模农民起义，导致隋朝灭亡。

第2课 唐朝建立与“贞观之治”

【必背重点】

  1. 唐朝建立：618年，唐高祖李渊建立唐朝，定都长安。
  2. 贞观之治：唐太宗李世民励精图治，开创了贞观之治。
  3. 虚心纳谏：“以人为镜，可以明得失”反映出唐太宗李世民虚心纳谏的特点，敢于直言进谏的魏征就是他可以“明得失”的人镜。
  4. 以民为本：“为君之道，必须先存百姓”“水能载舟，亦能覆舟”反映了唐太宗李世民以人为本的特点，为此他关心民间疾苦，减轻百姓负担，鼓励发展农业生产。
  5. 完善科举：唐太宗李世民进一步完善了科举制，增加科举考试科目，使进士科成为最重要科目。
  6. 殿试制度：一代女皇武则天开创了殿试制度，亲自面试考生，不拘一格选拔人才。
  7. 一代女皇：一代女皇武则天的统治“上承贞观，下启开元（治宏贞观，政启开元）”，在位期间经济继续发展，社会相对安定，为开元盛世的出现奠定基础。

【扩展掌握】

  1. 唐太宗改革：唐太宗李世民在开创贞观之治的过程中推行一系列改革，如进一步完善三省六部制，制定法律，减省刑罚，完善科举制等。
  2. 唐太宗用人：唐太宗李世民善用人才，“房谋杜断”指的是善于谋略的房玄龄和敢于决断的杜如晦，他们都是唐太宗手下的著名宰相。
  3. 唐朝教育：唐朝教育事业有很大发展，形成了一套从中央到地方、从官方到民间的教育体系，国子监是国家最高学府和教育行政机构。

第3课 “开元盛世”

【必背重点】

  1. 开元盛世：唐玄宗李隆基励精图治，在姚崇、宋璟等贤相辅佐下，开创了开元盛世。
  2. 新型农具：唐朝前期农业发展，出现了曲辕犁和筒车等新型农具。
  3. 国际都市：唐都长安是闻名中外的国际大都市，它“百千家似围棋局，十二街如种菜畦”，实行坊市制度，规划井然有序。

【扩展掌握】

  1. 手工业发达：蜀锦秀丽、青瓷如冰玉、白瓷类雪银、唐三彩精美等，都反映了唐代手工业发达。
  2. 唐朝商业：唐代商业繁荣，城市发展，草市逐渐兴起，陆上丝绸之路畅通，海上丝绸之路发展。
  3. 风气开放：唐朝女子骑马、弈棋等反映出当时社会风气比较开放。

第4课 安史之乱与唐朝衰亡

【必背重点】

  1. 由盛转衰：755年，安禄山、史思明发动安史之乱，使得唐朝由盛转衰，形成藩镇割据局面。
  2. 黄巢起义：“他日我若为青帝”“冲天香阵透长安，满城尽带黄金甲”等诗句的作者黄巢领导黄巢起义，给唐朝统治以致命打击。
  3. 唐朝兴亡：618年，唐高祖李渊建立唐朝；907年，朱温篡唐，建立后梁，唐朝灭亡。
  4. 五代十国：唐朝灭亡后进入五代十国时期，这是唐末以来藩镇割据局面的延续，动乱不休。

【扩展掌握】

  1. 叛乱原因：唐玄宗李隆基后期贪图享乐，朝政腐败，军事上形成外重内轻局面，致使中央与地方力量对比失衡，是安史之乱爆发的主要原因。
  2. 中兴名将：安史之乱历时8年，对社会经济造成巨大破坏，最终在郭子仪等中兴名将的努力和回纥军队的援助下，763年才平叛成功。
  3. 孕育统一：五代十国时期，各地经济联系、文化交流日益密切，各族交往进一步加强，统一因素不断增长；后周周世宗柴荣改革，为日后北宋统一打下基础。

第5课 隋唐时期的民族交往与交融

【必背重点】

  1. 天可汗：“自古皆贵中华，贱夷狄，朕独爱之如一”表明唐太宗李世民实行开明的民族政策，他因此被西北各族遵奉为“天可汗”。
  2. 文成公主入藏：唐太宗李世民统治期间的641年，文成公主入藏，促进了吐蕃经济和社会的发展，增进了唐蕃（汉藏）的友好关系。
  3. 唐朝与吐蕃：唐朝与吐蕃的关系发展，以和亲与会盟为主，和亲表现为文成公主入藏和金城公主入藏，会盟表现为唐蕃会盟碑。
  4. 管辖西域：唐朝先后设置西域都护府和北庭都护府，管辖西域的天山南北地区。
  5. 方式总结：唐朝处理民族关系的方式多样灵活，概括起来主要有对突厥的平叛战争，对吐蕃的和亲会盟，对西域的设立机构，对回纥、渤海、南诏的册封等。

【扩展掌握】

  1. 回纥发展：唐玄宗时，回纥首领骨力裴罗统一漠北，唐玄宗册封他为怀仁可汗；后来回纥改名回鹘，发展为如今维吾尔族先民的主体。
  2. 册封首领：唐玄宗时，不仅册封西北回纥首领为怀仁可汗，也册封东北靺鞨建立的渤海国首领大祚荣为渤海郡王，册封西南南诏首领皮逻阁为云南王。
  3. 唐与突厥：630年，唐太宗李世民击败东突厥汗国，唐高宗李治时，唐朝灭西突厥汗国，744年，唐朝平定了后突厥汗国，从此突厥在我国北方退出历史舞台。
  4. 唐朝统治：唐朝时期，边疆得到开发，中央王朝加强了对各族地区的统辖，统一多民族国家得到进一步发展和巩固。
  5. 唐朝疆域：唐朝是当时世界上版图最大、势力最强的国家，疆域东到大海，西到咸海，北达贝加尔湖，南及南海，东北至黑龙江以北、外兴安岭一带。

第6课 隋唐时期的中外文化交流

【必背重点】

  1. 玄奘鉴真：隋唐时期，为中外文化交流作出突出贡献的著名僧人是玄奘和鉴真。
  2. 玄奘西行：唐朝初期，玄奘西行天竺取经，为中国佛教的发展作出重大贡献，写成《大唐西域记》一书，是研究中外交流史的珍贵文献。
  3. 鉴真东渡：盛唐时期，鉴真东渡日本弘扬佛法，对中日文化交流作出了卓越贡献，是唐朝与日本文化交流中最有影响的人物。
  4. 人物精神：玄奘西行，长途跋涉，鉴真东渡，多次受阻，但他们都不畏艰险，矢志不渝，最终取得了成功。
  5. 交往特点：唐朝的对外交往具有“大进大出”的特点，既输出文化影响世界，有兼容并包，积极吸收外来文化，呈现出双向交流、开放包容的特点。

【扩展掌握】

  1. 对外开放：隋唐时实行对外开放的政策，日本、新罗等派遣大批使者、留学生等前往中国学习，这些遣唐使从中国返回后，推动了本国的政治改革，带回了中国的先进文化。
  2. 日本晁衡：日本遣唐使团中的杰出人物是阿备仲麻吕，他取汉名晁衡，与诗人王维、李白关系很好，李白诗“浮云不归沉碧海，浮云愁色满苍梧”悼念的就是他。
  3. 新罗崔致远：新罗派往中国的人中，最著名的是崔致远，他通过科举考试成为唐朝官员，用汉字写下大量诗文，收录入自己的诗文集《桂苑笔耕集》中。
  4. 与西方的交往：唐朝时，西方的阿拉伯帝国（大食）、东罗马帝国都与唐朝交往频繁，中国的造纸术通过阿拉伯帝国（大食）传播到西方。
  5. 唐人声誉：“九天阊阖开宫殿，万国衣冠拜冕旒”反映的正是唐朝的对外开放，当时唐朝与亚欧非等70多个国家和地区有交往，繁荣的盛唐声威卓著，中国人因此被称为“唐人”。

第7课 隋唐时期的科技与文化

【必背重点】

  1. 药王医德：隋唐时期在医药学领域最突出德人物是孙思邈，他被尊为“药王”，所著《千金方》以“人命至重，有贵千金，一方济之，德逾于此”为理念，体现出以人为本的高尚医德。
  2. 雕版印刷：雕版印刷品《金刚经》等文物表明，唐朝时期已经出现了雕版印刷术。
  3. 天文成就：隋唐时期在天文学领域最突出的人物是僧一行，他组织实测丢球子午线长度，制定《大衍历》以指导农业生产，对天文与数学的发展具有重要价值。
  4. 盛唐诗歌：唐朝是我国历史上诗歌创作的黄金时期，诗仙李白、诗圣杜甫等，都是唐朝诗歌的代表人物。
  5. 诗圣杜甫：杜甫的诗忧国忧民，悲悯众生，很多诗歌反映出政治腐败和战乱带给人民的痛苦，具有极高的史料价值，因此他本人被尊为“诗圣”，他的诗歌有“诗史”之称。

【扩展掌握】

  1. 诗仙诗魔：诗仙李白的诗具有浓郁的浪漫情怀，诗魔白居易的诗通俗易懂。
  2. 唐朝思想家：唐朝思想家的代表是韩愈和柳宗元：其中韩愈抨击佛教，主张复兴儒学；柳宗元是唐朝无神论者和朴素唯物主义思想的杰出代表。
  3. 书法绘画：隋唐时期书法绘画名家辈出，“颜筋柳骨”指的是书法家颜真卿和柳宗元；“吴带当风”指的是绘画风格奔放的“画圣”吴道子。
  4. 代表作品：颜真卿《颜勤礼碑》中的字端正劲美，雄浑敦厚；柳公权《玄秘塔碑》中的字方折峻丽，笔力劲健；吴道子的画《送子天王图》落笔雄劲，风格奔放；阎立本的画《步辇图》中人物形神兼备。
  5. 桥梁雕塑：甘肃敦煌莫高窟是闻名世界的艺术宝库，以唐朝开凿的石窟数量最多、保存最完成；隋朝李春设计并主持建造的赵州桥是世界上现存最古老的石拱桥之一。

第8课 北宋的政治

【必背重点】

  1. 北宋建立：960年，宋太祖赵匡胤黄袍加身，发动陈桥兵变，建立北宋，定都开封。
  2. 五代终结：979年，宋太宗赵光义灭掉北汉，结束五代十国分裂局面。
  3. 崇文抑武：针对唐末五代武将专权、政权动荡的积弊，北宋实行崇文抑武方针。
  4. 杯酒释兵权：宋太祖赵匡胤采用杯酒释兵权的方式，解除禁军高级将领的兵权。
  5. 中央集权：宋太祖赵匡胤在中央采取分化事权的方法，削弱相权，加强君权；在地方设通判限制知州权力。
  6. 改革科举：宋朝改革发展了科举制，大幅度增加科举取士名额。
  7. 崇文抑武积极作用：崇文抑武方针下，重用文臣，限制武将，扭转了五代十国尚武轻文的风气，有效避免了武将割据造反，稳固了政权。
  8. 王安石变法：1069年，为了摆脱统治危机，实现富国强兵，王安石开始变法。

【扩展掌握】

  1. 文臣管军务：崇文抑武方针下，军事方面，宋朝任用文臣管理军务，经常调换军队将领，定期换防，限制武将。
  2. 地方用文臣：崇文抑武方针下，地方治理方面，宋朝吸取五代武将控制地方的教训，用文臣担任地方长官。
  3. 经济措施：经济方面，宋朝加强对地方税收的控制，取消节度使收税权力，消除地方割据的物质基础。
  4. 科举作用：科举制度的发展，为读书人进入仕途开辟了道路，使北宋政权基础进一步扩大。
  5. 社会影响：宋朝在全国范围内营造浓厚的读书风气，促进了整个社会文化素养的提高，造就了宋朝科举发达、文化昌盛、人才辈出的文治局面。
  6. 崇文抑武的军事弊端：文臣掌兵，不熟军务，导致军队战斗力低下，在边疆战事中，北宋屡战屡败。
  7. 崇文抑武的政治弊端：崇文抑武，发展科举，加强皇权，使得政府机构重叠，相互牵制，官吏冗余，办事效率低下，财政负担大。
  8. 崇文抑武的社会弊端：崇文抑武下，逐渐出现了大地主、大官僚集团，他们和其他人一起兼并土地，致使社会问题严重，北宋出现统治危机。
  9. 变法内容：王安石变法设计经济、军事和教育等领域，军事领域如保甲法，经济领域如青苗法、募役法和方田均税法等。
  10. 变法失败：变法中的募役法限制了权贵特权，方田均税法下官僚地主无法逃避赋税，这些损害了大地主和权贵的利益，最终在他们的阻碍下变法失败。

第9课 辽、西夏与北宋并立

  1. 契丹建辽：916年，辽太祖耶律阿保机统一契丹各部，建立辽朝。
  2. 党项西夏：1038年，党项首领元昊建立西夏。
  3. 澶渊之盟：北宋与辽前期战和不定，最终于1005年签订“澶渊之盟”，保持长久和平。
  4. 文物证史：契丹货币、西夏货币都采用中原的圆形方孔样式，表明当时民族关系进一步发展，少数民族学习中原文化，各民族经济交流加强。
  5. 边境贸易：北宋与辽、西夏并立后，商业往来通过榷场进行，这促进了经济文化交流，加深了各族人民的沟通和了解，促进了民族交融，发展了民族关系。

【扩展掌握】

  1. 契丹发展：契丹的发展过程中，统一与汉化影响巨大，汉化使得契丹逐渐从事农耕，学会冶铁，提高了生产力，建立政权后更是创制了契丹文字；耶律阿保机统一契丹各部后，正式建立政权，定都上京。
  2. 燕云十六州：燕云十六州位置重要，是中原地域游牧民族南下的形胜之地，石敬瑭割让给契丹后，辽与中原王朝冲突加剧且大多数都占据优势。
  3. 宋夏议和：西夏与北宋先是军事冲突，北宋屡战屡败，西夏无法久战，最终议和。
  4. 花钱买和平：澶渊之盟后，北宋每年给辽钱物，称为“岁币”；宋夏议和后，北宋给西夏钱物，称为“岁赐”，但本质都是花钱买和平。
  5. 议和弊端：北宋崇文抑武，花钱买和平，辽国接收岁币，二者忽视了军事的发展，最终被崛起的新兴势力所攻灭。

第10课 金与南宋对峙

【必背重点】

  1. 女真建金：1115年，金太祖完颜阿骨打在完成女真各部统一的基础上建立金朝。
  2. 吞辽灭宋：1125年，金灭辽，1127年，金灭北宋，靖康之变。
  3. 岳飞抗金，精忠报国：金军南下烧杀抢掠，岳飞率岳家军抗击金军，军纪严明，取得“郾城大捷”，保护人民，深受人民爱戴、尊崇与怀念。
  4. 南宋偏安：“山外青山楼外楼……直把杭州作汴州”讽刺了南宋朝廷偏安江南的苟安心态。

【扩展掌握】

  1. 女真发展：女真崛起的过程中，汉化和统一影响巨大：女真崛起和金朝建立的前提是完颜阿骨打统一了女真各部；金朝建立后学习中原制度，颁行女真文字，提倡儒学，推行科举，学习中原文化。
  2. 北宋灭亡原因：北宋的灭亡，统治者的昏庸无能要负很大责任，人民奋勇抗金时，懦弱无能的宋徽宗赵佶、宋钦宗赵桓只想着妥协求和，最终酿成了靖康之变。
  3. 宋辽兴亡：辽建立于916年，灭亡于1125年；北宋建立于960年，灭亡于1127年。
  4. 南宋君臣：1127年，宋高宗赵构建立南宋；1141年，赵构秦桧君臣对金称臣、割地、送岁币，杀害岳飞，偏安江南，绝对的昏君佞臣。
  5. 千古评价：“赖有岳于双少保，人间始觉重西湖”歌颂了岳飞精忠报国，为国为民的精神，值得后人永远怀念。
  6. 千古骂名：“青山有幸埋忠骨，白铁无辜铸佞臣”中的佞臣之首是谋杀岳飞的秦桧，它的铁铸像至今仍跪祭祀岳飞的岳王庙中，这是人民对忠奸最直观朴素的评价。

第11课 金与南宋对峙

【必背重点】

  1. 一代天骄：1206年，成吉思汗铁木真统一蒙古草原，建立蒙古政权。
  2. 大哉乾元：1271年，忽必烈建立元朝，1276年，元灭南宋，1279年，元朝完成统一。
  3. 铁血丹心：文天祥写下“人生自古谁无死，留取丹心照汗青”的千古名句，拒不降元，英勇就义，体现了坚韧不屈的民族气节。
  4. 疆域广袤：“自封建变为郡县，有天下者……幅员之广，咸不逮元”表明元朝是我国历史上版图最大的朝代，它把今天西南的西藏、云南，东南的台湾等都纳入中央管辖。
  5. 行省制度：“陕西行省、四川行省、江西行省”等表明，元朝在地方开创了行省制度，这提高了行政效率，成为我国省制的开端。
  6. 边疆治理：元朝因地制宜治理边疆，设澎湖巡检司加强对台湾的管辖，正式把台湾纳入中央管辖；设宣政院管辖西藏，正式把西藏纳入中央管辖。
  7. 民族融合：元朝民族融合进一步加强，出现了新的民族——回族（回回），各族以中原文化为主的文化认同趋向加强。

【扩展掌握】

  1. 蒙古扩张：蒙古统一崛起后，发动大规模对外扩张战争，1227年灭西夏，1234年灭金，向西打到欧洲多瑙河流域。
  2. 蒙古汉化：忽必烈继承蒙古帝国汗位后，接收汉人儒臣建议，“行仁政”“行汉法”“不嗜杀”，施行治国安民的方略，为元朝的建立和统一奠定了基础。
  3. 定都北京：1272年，元朝定都大都，即今天的北京，元明清三朝都在北京定都。
  4. 中央集权：元朝在中央设中书省掌管全国行政事务，设御史台负责监察事务，设枢密院负责全国军政事务，逐步确立中央集权制度。
  5. 驿传制度：元朝疆域空前辽阔，为加强对地方管辖，及时传递政令，元朝在全国普遍设置驿站，实行驿传制度，这对维护国家统一，促进经济、文化交流起了的重要作用。
  6. “人迹所至，皆置驿传，使驿往来，如行国中”反映的正是元朝普遍设置驿站，实行驿传制度的史实。
  7. 管辖西北：元朝在西北地区设置北庭都元帅府等机构，官吏西域军政事务。
  8. 元朝统一对民族交融的影响：元朝境内大规模的人口流动，促进了各族经济、文化的交流和发展，加强了各族间的了解与联系，对中华民族进一步交融具有重要意义。
  9. 四等人制：元朝统治者为了维护自己的统治利益，实行“四等人制”，对不同民族采取差别对待的措施，具有民族歧视性。
  10. 元朝灭亡：元朝后期，民族矛盾减弱，社会贫富分化带来的阶级矛盾突出，最终导致元朝于1368年灭亡。

第12课 辽宋夏金元时期经济的繁荣

【必背重点】

  1. 经济重心南移：“国家根本，仰给东南”“苏湖熟，天下足”反映的历史现象是经济重心南移，这一过程在南宋正式完成。
  2. 经济重心南移表现：经济重心南移完成在人口方面的表现为南方人口总量与增长速度均超过北方；在农业方面的表现为南方农业发展，江南地区供应政府粮食等。
  3. 经济重心南移原因：经济重心南移的原因与江南开发的原因基本一致，即北方人口南迁促进了南方发展，南方自然条件优越，社会相对稳定，南方统治者重视经济发展，制定合理政策等。
  4. 纸币出现：北宋前期四川地区出现的“交子”是世界上最早的纸币，纸币的产生既是商业发展对货币交易需求增长的必然结果，也有利于商业的发展。
  5. 宋代农业发展：宋元农业发展表现在：高产作物——占城稻的引进，新式农具——秧马的出现等。
  6. 市井文化发展：宋元时期，随着城市繁荣，人们的文化逐渐丰富，出现了娱乐兼营商业的场所——瓦子，瓦子中也有专供演出的圈子——勾栏。

【扩展掌握】

  1. 宋代农业发展：宋元农业发展还表现在：耕种技术——复种的推广，南粮北运交通——元朝大运河和海上航线的开辟等。
  2. 经济作物种植：宋元南方经济作物的发展很大，体现在茶树的普遍种植和棉花种植区域的北扩等方面。
  3. 宋代手工业发展：宋代手工业发展中，因为黄道婆的贡献，棉纺织业兴起，棉布逐渐替代麻布。
  4. 宋代瓷器发展；宋代是我国瓷器发展史上的辉煌时代，江南成为制瓷中心，景德镇后来发展为瓷都。
  5. 宋代商业的发展：宋元商业繁荣超过前代，具体表现在大型城市如北宋东京（开封），南宋临安（杭州），元大都（北京）的出现，早市和夜市的进一步发展，坊市制度的打破等。
  6. 史料印证宋代商业发展：“夜市直至三更尽，才五更又复开张，耍闹去处，通宵不绝”反映出宋代商业打破了时间和空间的限制，时间上夜市和早市进一步发展，空间上坊市制度限制取消等。
  7. 文物印证宋元社会：北宋张择端的《清明上河图》展现了宋代商业繁荣的盛况；宋代蹴鞠纹青铜镜反映了宋代市井文化的繁荣和娱乐活动的多样。
  8. 隋朝元朝大运河对比：隋朝大运河以洛阳为中心，元朝大运河截弯取直，直通北京。二者都贯通南北，加强南北沟通，便利了南粮北运。
  9. 辽夏金的经济：毛纺织业是西夏手工业的重要部门；辽金的制瓷技术相当进步；辽的铁制农具制作水平较高，西夏兵器制作精良；金造出了撞冰船；辽的上京、西夏兴庆府、金的燕京商业繁荣，金在部分地区对往来商旅征收商税。

第13课 辽宋夏金元时期的对外交流

【必背重点】

  1. 市舶司与海外贸易：“市舶之利最厚……所得动以万计”表明，宋元时期，政府鼓励海外贸易，在主要港口设立市舶司以管理海外贸易，通过外贸税收补充政府财政收入。
  2. 宋元海贸发达原因：政府重视，设市舶司管理；经济重心南移，南方手工业发达，商业繁荣；造船技术的进步和指南针应用于航海；海外交通的发展等。
  3. 宋元时期的文化交流：宋元时期中外文化交流密切，具有双向交流的特点，中国的四大发明等输往国外，影响深远；中国也吸收西方科技文化等。

【扩展掌握】

  1. 宋代交通：宋朝只是局部统一，与西域、中亚的陆路交通被阻断，依靠经济中心南移，造船技术与航海技术的进步，发展了海上交通，形成多条航线，最远可达波斯湾及东非海岸。
  2. 元代交通：元朝建立后，完成统一，疆域广袤，陆路和海路交通的范围进一步扩大，开创了中外交通的新局面，海上丝绸之路进入鼎盛时期。
  3. 大商港：宋元时期，海外贸易发达，广州、泉州、明州等都是闻名世界的大商港，海外贸易繁荣，造船业发达。
  4. 宋元海贸发达表现：宋元时期的海外贸易超过前代，第一体现在商船航行远，远大阿拉伯半岛和非洲东海岸；第二体现在贸易国家地区多，宋代时为五六十，元代时超过140；第三体现在商品种类多。
  5. 辽夏金的交通与贸易：辽夏金通过草原丝绸之路维持着与西域、中亚较大规模的交流；辽金与朝鲜半岛保持经济上的练习，与日本、波斯、大食（阿拉伯）等国存在贸易关系。

第14课 辽宋夏金元时期的科技与文化

【必背重点】

  1. 活字印刷术：北宋的毕昇发明了活字印刷术，改革了唐朝就已出现的雕版印刷术，提高了印刷效率。
  2. 指南针应用航海：北宋末期，指南针开始应用于航海，影响深远。
  3. 指南针对世界的影响：指南针经阿拉伯传入欧洲后，为欧洲新航路开辟和环球航行提供保证，极大推动了世界经济、文化的交流与发展。
  4. 火药用于军事：唐朝时我国发明了火药，唐末开始运用到军事领域，宋元时期在军事上的运用更加广泛。
  5. 火药对世界的影响：火药经阿拉伯传入欧洲后，对欧洲火器制造和作战方式产生影响，推动了欧洲社会变革。
  6. 宋词：宋词是宋代主要的文学形式之一，宋代杰出词人有词风豪迈飘逸的苏轼、女词人李清照和词风豪放的辛弃疾。
  7. 元曲：元朝元曲中，最优秀的代表人物是关汉卿，他的代表作是《窦娥冤》。
  8. 《资治通鉴》：“鉴于往事，有资于治道”反映的是史书是《资治通鉴》，它是一部编年体通史，记述了从战国到五代的历史。
  9. 司马光：《资治通鉴》的作者是北宋司马光，他与《史记》作者司马迁并称“史学两司马”。
  10. 《史记》与《资治通鉴》对比：《史记》是纪传体通史，《资治通鉴》是编年体通史，二者共同记录的内容是从战国到汉武帝时期的历史。

【扩展掌握】

  1. 天文历法：北宋沈括创制了“十二气历”，有利于指导农业生产；元代郭守敬编成《授时历》，取“敬授民时”之意，对知道农业生产有很大帮助。
  2. 中国历法：中国历法都具有实用性，最大的作用之一是指导农业生产。
  3. 《梦溪笔谈》：北宋科学家沈括撰写的《梦溪笔谈》是科技著作，综合了我国古代的许多科技成就。
  4. 精准历法：郭守敬的《授时历》非常精准，与现行公历的一年取值几乎相同，但比现行公历早约300年。
  5. 程朱理学：宋代儒学在兼收并蓄的基础上形成理学，其成就突出者是北宋程颐、程颢，南宋朱熹。
  6. 理学束缚：朱熹是理学的集大成者，他强调存天理、灭人欲，要求人们遵从传统伦理道德，这强调封建礼教，有利于维护封建统治，逐渐称为我国古代的统治思想，束缚了人们的精神追求，扼杀了科技发展。
  7. 宋元书画：宋元时期书法绘画艺术成就很高，宋代书法家苏黄米蔡指的是苏轼、黄庭坚、米芾（fu）、蔡襄，元代著名书画家赵孟頫是我国古代四大楷书“颜柳欧赵”之一。

第15课 明朝的统治

【必背重点】

  1. 明朝建立：1368年，明太祖朱元璋建立明朝，定都应天府（今南京），随后灭元。
  2. 强化皇权：为了强化皇权，在中央，明太祖废除丞相；在地方，他废行省设三司，分封诸子为王；在监察方面，他设立锦衣卫监督官民。
  3. 八股取士：明朝在科举方面实行八股取士，严格限制考试的内容与格式，禁锢思想。
  4. 管辖东北：在东北，明成祖朱棣设奴儿干都司，加强对东北女真等部的直接管理，把黑龙江、乌苏里江、松花江流域等地和库页岛纳入有效管理。

【扩展掌握】

  1. 明朝集权特点：明太祖朱元璋认为元朝灭亡是由于 地方分权和朝臣权力过大，所以他中央废除丞相，地方废行省设三司，让中央和地方各部门既互不统属，又互相监督，使皇权高度集中，君主专制大为加强。
  2. 迁都北京：明成祖朱棣通过“靖难之役”夺取皇位，1421年迁都北京，加强北方防御。
  3. 东厂西厂：在监察方面，除了明太祖朱元璋设置锦衣卫外，后来的明成祖朱棣还设立了东厂，明宪宗成立了西厂，这些都是皇帝的耳目和爪牙，负责监督官民。
  4. 隆庆和议：明朝隆庆年间，俺答封贡，隆庆和议，终结明朝和蒙古敌对状态。

第16课 明朝的对外关系

【必背重点】

  1. 郑和下西洋：1405-1433年，明朝的郑和七下西洋，最远到达非洲东海岸和红海沿岸，规模大，时间长，航程远，堪称世界航海史上的壮举，为人类航海事业作出伟大贡献。
  2. 郑和下西洋特点：郑和下西洋是和平交流之旅，宣扬明朝国威但不称霸，增进了中国与亚非国家和地区的了解与友好往来。
  3. 郑和下西洋的经济属性：虽然郑和下西洋的主要目的是政治上宣扬明朝国威，但也在经济上推动了中外贸易的发展，发展了海上丝绸之路。
  4. 戚继光抗倭：明朝戚继光抗倭基本解除了东南沿海的倭患，他领导的抗倭战争是一场反侵略的战争，维护了人民的生命和财产安全，他是当之无愧的民族英雄。
  5. 戚继光品质：“封侯非我意，但愿海波平”对应的历史事件是戚继光抗倭，这句诗体现了戚继光爱国护民、淡泊名利的品质。

【扩展掌握】

  1. 抗倭援朝：明朝万历年间两次抗倭援朝，沉重打击了日本的政治军事实力，但明朝国力也因此受到削弱，对东亚政治军事格局产生深远影响。
  2. 葡萄牙非法攫取澳门租住权：明朝时，西方欧洲殖民者来我国沿海进行侵略活动，其中葡萄牙人通过非法手段攫取在澳门的租住权。
  3. 澳门主权归属问题：澳门是中华不可分割的一部分，明朝是，虽然葡萄牙人攫取了租住权，但主权一直属于中国，明朝政府设守澳官并驻军，加强对澳门管理。

第17课 明朝的灭亡和清朝的建立

【必背重点】

  1. 张居正改革：明朝中后期，政治腐败，张居正进行“一条鞭法”改革，取得积极成效，但最终在张居正死后改革失败。
  2. 李自成起义：明朝末年，李自成领导农民起义，提出“均田免粮”口号，严明军纪，发展壮大，1644年建立大顺政权，灭亡明朝。
  3. 清军入关：1644年明朝灭亡后，山海关守将吴三桂降清，清军入关，击败李自成，迁都北京，逐步建立对全国的统治。

【扩展掌握】

  1. 明朝腐败：明朝中后期的政治腐败体现在多个方面：皇帝沉迷享乐，重用宦官，疏于朝政；统治集团内容不勾心斗角，结党营私；各级官吏贪赃枉法，盘剥百姓等。
  2. 明朝社会危机：明朝政治腐败造成中央对社会的控制力不断下降，皇族、宦官、官僚、地主等疯狂兼并土地，致使大量农民流离失所。
  3. 张居正改革内容：张居正改革被称为“一条鞭法”，基础是清丈田亩，合并税收，计入田亩，以白银收税，这简化了税收手续，减少了地方官吏欺上瞒下渔利百姓的机会，减轻百姓负担的同时增加政府税收。
  4. 农民起义背景：张居正改革失败后，明朝政治更加腐败；明末天灾不断，百姓生活困难；政府财政危机严重，不断加派赋税：这些都使得百姓无法忍受，最终爆发大规模农民起义。
  5. 女真崛起：明朝后期，东北女真崛起，努尔哈赤于1616年基本统一女真各部，建立“后金”，1635年，皇太极改女真族名为满洲，1636年，改国号为大清。
  6. 清朝统治：1644年清朝入关，迁都北京后，为了巩固统治，政治上沿袭中原王朝做法，进一步加强中央集权，维护大一统，思想文化上推崇儒家学说。

第18课 清朝的边疆治理

【必背重点】

  1. 郑成功收复台湾：1662年，民族英雄郑成功从荷兰侵略者手中收复宝岛台湾，维护了祖国统一和领土完整。
  2. 台湾建制：1684年，清朝在东南台湾设台湾府，加强了中央对台湾的管辖，巩固了东南海防，促进了台湾社会经济的发展。
  3. 管辖西藏：在政治上，清朝1727年设驻藏大臣，此后不断提升驻藏大臣权力，一步步加强对西南西藏的管辖。
  4. 管辖西藏：在宗教事务上，清政府通过册封达赖、班禅等宗教领袖，设立金瓶掣签制度，加强对西南西藏的管辖。
  5. 管辖新疆：在政治上，清朝设立伊犁将军，加强对西北新疆管辖。
  6. 维护统一：在西北新疆，面对叛乱，康熙平定噶尔丹叛乱，乾隆平定大、小和卓叛乱，维护统一。
  7. 万里归国：1771年，西迁的土尔扈特部，在首领渥巴锡的带领下万里归国，被清政府妥善安置在新疆，为统一多民族国家的巩固和发展谱写了光辉篇章。
  8. 东北抗俄：面对沙俄对东北的侵略，康熙组织雅克萨之战，击败俄军。
  9. 条约定疆：1689年，中俄签订《尼布楚条约》，从法律上明确黑龙江和乌苏里江流域包括库页岛在内的东北广大地区都是中国领土。
  10. 边疆治理总结：清朝前期对边疆治理的方式多样灵活，因地制宜，有设立机构、册封、创新制度、对内平定叛乱、对外抗击侵略、签订边界条约等。
  11. 积极影响：清朝前期对边疆的有效治理，维护了国家统一，加强了对边疆的管辖，奠定了中国现在版图的基础；因地制宜的特点，加强了各民族之间的经济文化联系，促进了边疆地区的开发，巩固和发展了统一多民族封建国家。
  12. 文物证史：达赖金印、班禅金印、金瓶掣签的金奔巴瓶反映了清朝对西南西藏地区的管辖；新疆伊犁将军府反映了清朝对西北新疆地区的管辖；神威无敌大将军铜炮反映了清朝在东北抗击沙俄的雅克萨之战。

【扩展掌握】

  1. 册封宗教领袖：清朝顺治时册封了达赖喇嘛，康熙时册封了班禅额尔德尼，此后历代达赖和班禅都必须经过中央政府的册封，这在一定程度上保证了达赖、班禅等在西藏影响力巨大的宗教领袖心向中央，维护统一。
  2. 中央权力的强化：清政府1727年设驻藏大臣监督西藏地方政务，1751年设噶厦授予驻藏大臣管理西藏政教事务的权力，1793年颁布《钦定藏内善后章程》提升驻藏大臣地位：驻藏大臣权力的增长表明中央政府对西藏管辖的逐步加强。
  3. 清朝疆域：清朝全盛时期疆域广阔，奠定了我国今天的疆域基础，她西跨葱岭，西北至巴勒喀什池，北接西伯利亚，东北至黑龙江以北的外兴安岭和库页岛，东临太平洋，东南到台湾及其附属岛屿，包括钓鱼岛、赤尾屿等，南至南海诸岛，西南达喜马拉雅山脉。

第19课 清朝君主专制的强化

【必背重点】

  1. 清朝集权：清朝雍正帝设立军机处，使皇帝具有至高无上的权威，君主集权达到顶峰。
  2. 闭关锁国：“禁海令”，广州一口通商、广州十三行统一经营对外贸易反映出清朝实行闭关政策，这使得中国逐渐落后于世界潮流。
  3. 文字狱：因诗句“清风不识字，何故乱翻书”“一把心肠论浊清”而被杀的情况表明清朝实行文字狱，这禁锢了思想，组在了思想、学术的发展进步。
  4. 白莲教起义：1796年，白莲教起义爆发，是清朝走向衰落的转折点。

【扩展掌握】

  1. 奏折制度：康熙时期，出现奏折制度，官员秘密向皇帝呈送报告，这使君主专制进一步强化。
  2. 君主专制的强化：清朝初期存在议政王大臣会议制约皇权，康熙设南书房，雍正设军机房，绕过议政王大臣会议处理军国大事，不断削弱议政王大臣会议的权力，不断强化君主专制。
  3. “（军机大臣）只供传述缮撰，而不能稍有赞画于其间”表明军机处只能传达抄写皇帝旨意，不能发表自己的意见，这便于皇帝独掌朝政，君主专制达到顶峰。
  4. 文化专制政策：清朝为了维护集权统治，表面上尊孔读经，大规模整理文献、编纂书籍，修成《古今图书集成》《四库全书》等，实际上实行文化专制政策，全面检查禁毁书籍，摧残文化。
  5. 社会危机：“和珅跌倒，嘉庆吃饱”反映出清朝君主专制下，政治腐败，官吏贪污横行；人口快速增长导致人地矛盾突出，土地兼并加剧，社会贫富分化现象严重。
  6. 西方侵略：葡萄牙人攫取澳门租住权、荷兰侵占台湾等表明，明清时期，西方殖民者在向东方扩展势力。
  7. 闭关目的：清政府担心国家领土主权收到侵犯，更担心沿海人民同外国人交往会危及自己统治，所以实行闭关政策。
  8. 对外贸易：广州一口通商，广州十三行统一经营对外贸易表明，清朝闭关政策下不是禁止对外贸易，而是严格限制对外贸易。

第20课 明清时期社会经济的发展

【必背重点】

  1. 农业发展：明清农业发展的表现之一是原产于美洲的高产作物玉米、甘薯、花生等传入并得到推广，极大地提高了粮食产量。
  2. 手工业发展：明清时期手工业在前代的基础上继续发展，景德镇发展为制瓷中心（瓷都），出现了成熟的手工工场，提高了生产效率，手工业制品畅销海内外。
  3. 商业发展：明清时期商业发展的表现之一是出现具有地域特色的商帮，代表有山西晋商和徽州徽商，商帮的活动对当时社会经济的发展产生了巨大的推动作用。
  4. 史料证史：《盛世滋生图》反映了明清时期商业的繁荣；《洪武鱼鳞图册》《清朝开荒执照》等文物反映了明清初期对农业的重视，与核查土地、鼓励垦荒等史实印证。
  5. 明清人口增长原因：明清人口快速增长，在经济上得益于垦荒政策的推行，高产作物的传入和粮食产量的提高，在政治上得益于取消人口税的“摊丁入亩”等政策的实行。

【扩展掌握】

  1. 农业政策：明清初期，统治者都采取鼓励垦荒，轻徭薄赋的政策，重视经济发展，核查土地，兴修水利，这促进了经济的恢复和人口的增长。
  2. 资本主义萌芽：“机户出资，织工出力”反映明清时期在手工业生产方面出现雇佣关系，产生资本主义萌芽。
  3. 全球经济：明清时期中国经济上全球经济体系中的重要一环，清政府的闭关政策是严格限制海外贸易而非严厉禁止海外贸易，当时中国商品大量销往国外，美洲白银等通话海外贸易流入中国。
  4. 人口增长：明清时期，人口快速增长，人地矛盾日益突出，好的一面是大量人口流向边疆，促进了边疆的社会繁荣，坏的一面是土地兼并严重，贫富分化十分严重，社会危机重重。

第21课 明清时期的科技与文化

【必背重点】

  1. 医药著作：明朝医药学家李时珍编出《本草纲目》这部药物学著作，总结了我国古代药物学成就，丰富了我国医药学宝库。
  2. 农业著作：明朝科学家徐光启著有《农政全书》，全面总结了我国古代农业生产的先进经验、技术革新，是一部重要的农业科学巨著。
  3. 科技著作：明朝宋应星编著了《天工开物》这部科技名著，全面总结了我国古代农业和手工业的生产技术，被誉为“中国17世纪的工艺百科全书”。
  4. 撰写思想：卷分前后，贵五谷而贱金玉反映出宋应星以农为本的思想。
  5. 明清科技特点：《本草纲目》《农政全书》《天工开物》都是总结性科技著作，这从侧面反映了明清时期科技的原创性不足，封建君主专制制度达到顶峰，八股取士和文字狱、文化专制政策等都在一定程度上扼杀了科技创新。
  6. 明清小说：明清时期，商品经济发展，市民文化繁荣，使得小说、戏曲等大众化的文学艺术形式有了突出发展。
  7. 明清戏剧：明清时期戏曲的代表有昆曲和京剧，国粹京剧的形成过程中，徽班吸收了昆曲、秦腔等地方戏的优点，这反映出文艺创作应兼容并包，融合创新，不可故步自封。
  8. 万里长城：我们今天所看到的长城主要是明朝修筑的，明长城动起鸭绿江边，西至嘉峪关，是为了防御北方蒙古族南扰而修筑的完整防御体系。
  9. 明清北京城：明清北京城围绕中轴线对称展开，布局均衡，完整和谐，紫禁城居中等，都反映出封建君主专制的强化。

【扩展掌握】

  1. 明清思想家：明朝思想家王守仁提倡“心即理”“致良知”，主张“知行合一”；明末清初，社会动荡，思想活跃，黄宗羲反对君主专制，顾炎武认为“天下兴亡，匹夫有责”，王夫之认为社会是发展、进化的，是朴素唯物主义思想家。
  2. 昆曲经典：明朝汤显祖的《牡丹亭》，清朝洪昇的《长生殿》和孔尚任的《桃花扇》，都是昆曲的传世之作。
  3. 四大名著：四大名著中，以三国史实为基础的历史小说《三国演义》，以玄奘西行为原型创作的浪漫主义神话小说是《西游记》，以官逼民反为主题，描写梁山好汉的小说是《水浒传》，清代艺术成就最高，深刻反映封建社会末期社会现实和尖锐矛盾的小说是《红楼梦》。
  4. 文献证史：“为天下之大害者，君而已矣”反映出黄宗羲反对君主专制；做官“为天下，非为君也；为万民，非为一姓也”反映出黄宗羲批判“君为臣纲”的封建教条。
  5. 长城作用：长城处于北方游牧地区与农耕地区的连接线上，在作为防御军事的同时，也有利于少数民族（游牧）与汉族（农耕）的经济文化交流，称为各民族交往的纽带，称为中华民族精神的象征。
